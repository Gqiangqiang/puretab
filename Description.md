**PureTab：极简新标签页扩展**

PureTab 是一款极简浏览器扩展，将新标签页变为干净无干扰的仪表盘，兼顾美观、实用与隐私，提升专注力和效率。

**界面与核心功能**  
- **实时时钟** – 显示时间、日期和星期，配合柔和渐变背景。  
- **智能搜索栏** – 支持直接输入网址跳转，并可切换搜索引擎（Bing、百度、Google 或自定义）。  
- **可定制快捷链接** – 支持添加、编辑、删除、拖拽排序，右键可自定义图标大小和样式。

**个性化设置**  
- **主题切换** – 提供浅色、深色和自动模式，支持更换背景图片。  
- **锁屏功能** – 按 `Ctrl/Cmd+Shift+L` 手动锁定新标签页（时钟仍可见），无需关闭任何标签。

**隐私与数据持久化**  
- 不收集任何个人数据。  
- 所有设置和快捷方式均通过现代存储技术保存，跨会话保持一致体验。

**使用步骤**  
1. 从浏览器商店安装扩展。  
2. 打开新标签页，即可看到简洁界面、时钟和搜索栏。  
3. 直接搜索或输入网址访问。  
4. 通过拖拽或右键管理快捷链接。  
5. 随时切换主题或更换背景图。  
6. 离开时用快捷键锁定标签页。  
7. 所有偏好设置自动保存，下次打开依然生效。

**核心特点总结**  
- 极简无冗余设计  
- 实时时钟（含日期/星期）  
- 灵活可配的搜索栏  
- 快捷链接自由管理  
- 浅色/深色/自动主题 + 自定义背景  
- 手动锁屏模式  
- 设置持久化保存


**PureTab: Minimalist New Tab Extension**  

PureTab is a minimalist browser extension, it transforms your new tab into a clean, distraction‑free dashboard. It blends elegance, utility, and privacy to boost focus and productivity.  

**Interface & Core Features**  
- **Real‑time clock** – displays time, date, and weekday with a smooth gradient background.  
- **Smart search bar** – supports URL navigation and configurable search engines (Bing, Baidu, Google, or custom).  
- **Customizable quick links** – add, edit, remove, reorder via drag‑and‑drop, and right‑click for icon personalization (size and style).  

**Customization**  
- **Themes** – toggle between light, dark, or auto‑schedule modes; replace background images freely.  
- **Lock screen** – press `Ctrl/Cmd+Shift+L` to manually lock the new tab (clock remains visible) without closing any tabs.  

**Privacy & Persistence**  
- No personal data collected.  
- All settings and shortcuts are saved persistently across sessions using modern storage, ensuring a consistent experience every time.  

**How It Works**  
1. Install the extension from your browser’s store.  
2. Open a new tab to see the sleek interface with clock and search.  
3. Search or enter URLs directly.  
4. Manage shortcuts via drag‑and‑drop or right‑click.  
5. Switch themes or background images at any time.  
6. Lock the tab with the keyboard shortcut when stepping away.  
7. Enjoy automatic saving of all preferences for future sessions.  

**Key Features Recap**  
- Minimalist, clutter‑free design  
- Live clock with date/weekday  
- Powerful, configurable search bar  
- Flexible quick‑link management  
- Light/dark/auto themes + custom backgrounds  
- Manual lock screen mode  
- Persistent storage for settings
